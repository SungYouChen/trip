這是您的蜜月旅行行程網站專案。

## 🌐 您的專屬網站
- **永久網址**: [https://trip.elk-web.com](https://trip.elk-web.com)
- **本地網址**: [http://localhost:8000](http://localhost:8000)

## 🚀 如何啟動網站
為了讓外網能連上您的網站，請務必按照以下步驟啟動：

1. **打開終端機 (Terminal)**
2. **進入資料夾** (如果您還沒進入)：
   ```bash
   cd ~/Desktop/project/Japan1141228-1150108
   ```
3. **執行啟動腳本**：
   ```bash
   ./share_app.sh
   ```

## ⚠️ 注意事項
1. **不要關閉視窗**：啟動後，Terminal 視窗需要保持開啟，網站才能運作。
2. **電腦不能睡眠 (重要！)**：
   - 網站是跑在您的電腦上的，所以如果電腦進入睡眠 (Sleep Mode)，網站就會斷線。
   - **設定方法**：
     - 打開「**系統設定 (System Settings)**」
     - 點選「**顯示器 (Displays)**」或「**能源節約器 (Energy Saver)**」
     - 找到「**避免電腦在顯示器關閉時自動睡眠**」並開啟它。
     - 或者：把「**關閉顯示器**」的時間設為「**永不 (Never)**」。
   - **進階技巧 (推薦)**：您可以下載一個免費的小軟體叫 **"Amphetamine"**，它可以一鍵讓電腦保持清醒。
   - **快速關螢幕指令**：設定好不休眠後，您可以執行此指令來立即關閉螢幕 (省電)：
     ```bash
     pmset displaysleepnow
     ```
